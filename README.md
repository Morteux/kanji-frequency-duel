# Most frenquent kanji
Just pick one — I’m pretty sure you’ve got no idea which one is more common.

Roughly 777 characters cover 98% of written Japanese, while 1,477 characters provide 99.9% coverage.

## Key Frequency Data and Milestones
- 100-300 Kanji: Covers basic signs, headlines, and common components (e.g., 日, 一, 国, 会).
- 1,000 Kanji: Allows for functional reading of daily material and manga.
- 2,000+ Kanji: Covers 90% or more of, newspapers, covering nearly all of the 2,136 Jōyō (regular use) kanji.

## Top 10 Most Frequent Kanji (Approximate Order) 
- 日 (sun, day)
- 一 (one)
- 国 (country)
- 会 (meet, association)
- 人 (person)
- 年 (year)
- 大 (big)
- 十 (ten)
- 二 (two)
- 本 (book, origin)
